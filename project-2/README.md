# Python Quiz Game

## 1. 프로젝트 개요
이 프로젝트는 Python으로 만든 콘솔 기반 퀴즈 게임입니다.  
사용자는 메뉴를 통해 퀴즈를 풀고, 새로운 퀴즈를 추가하고, 등록된 퀴즈 목록과 최고 점수를 확인할 수 있습니다.  
또한 프로그램을 종료한 뒤 다시 실행해도 퀴즈 데이터와 최고 점수가 유지되도록 `state.json` 파일에 저장하는 기능을 구현했습니다.

이 프로젝트를 통해 다음 내용을 직접 구현하고 확인하는 것을 목표로 했습니다.

- Python 기본 문법 활용
- 클래스와 객체를 통한 코드 구조화
- JSON 파일 저장을 통한 데이터 영속성
- Git과 GitHub를 활용한 버전 관리

---

## 2. 퀴즈 주제 선정 이유
퀴즈 주제는 **Python 기초 문법**으로 선정했습니다.

이유는 다음과 같습니다.

1. 이 프로젝트 자체가 Python 학습을 목표로 하기 때문에, 퀴즈 주제를 Python으로 정하면 학습과 구현을 동시에 진행할 수 있습니다.
2. 변수, 자료형, 조건문, 반복문, 함수 같은 기본 개념을 문제로 구성하기 쉬워 과제 요구사항에 적합합니다.
3. 직접 문제를 만들면서 Python 개념을 다시 정리할 수 있어 학습 효과가 높다고 판단했습니다.

## 4. 기능 목록

### 4-1. 메뉴 기능
프로그램 실행 시 다음 메뉴가 출력된다.

1. 퀴즈 풀기  
2. 퀴즈 추가  
3. 퀴즈 목록 보기  
4. 최고 점수 확인  
5. 종료  

사용자는 번호를 입력하여 원하는 기능을 선택할 수 있다.

---

### 4-2. 퀴즈 풀기
- 저장된 퀴즈를 순서대로 출제한다.
- 각 문제는 문제, 선택지 4개, 정답 번호로 구성된다.
- 사용자가 정답 번호를 입력하면 정답/오답 여부를 출력한다.
- 모든 문제 풀이 후 총 점수를 출력한다.
- 최고 점수보다 높은 점수일 경우 최고 점수를 갱신하고 저장한다.

---

### 4-3. 퀴즈 추가
- 사용자가 새로운 퀴즈를 직접 등록할 수 있다.
- 입력 항목:
  - 문제
  - 선택지 4개
  - 정답 번호 (1~4)
- 입력 검증을 수행하며, 잘못된 입력 시 재입력을 요구한다.
- 추가된 퀴즈는 `state.json`에 저장된다.

---

### 4-4. 퀴즈 목록 보기
- 현재 저장된 퀴즈의 문제 목록을 출력한다.
- 퀴즈가 없는 경우 안내 메시지를 출력한다.

---

### 4-5. 최고 점수 확인
- 현재 저장된 최고 점수를 출력한다.
- 퀴즈를 한 번도 풀지 않은 경우 안내 메시지를 출력한다.

---

### 4-6. 종료
- 프로그램 종료 전 현재 상태를 저장한다.
- 다음 실행 시 데이터를 유지한다.

---

## 5. 입력 처리 및 예외 처리

### 5-1. 숫자 입력 처리
- 입력값 앞뒤 공백 제거
- 빈 입력 방지
- 숫자 변환 실패 시 재입력
- 허용 범위 초과 시 재입력

예:
- 메뉴: 1 ~ 5
- 정답: 1 ~ 4

---

### 5-2. 강제 종료 처리
다음 예외 발생 시 안전하게 종료 처리한다.

- `KeyboardInterrupt (Ctrl + C)`
- `EOFError`

동작:
- 안내 메시지 출력
- 데이터 저장
- 정상 종료

---

### 5-3. 파일 예외 처리
- `state.json` 파일이 없을 경우 기본 퀴즈 생성
- 파일 손상 시 기본 데이터로 복구
- 읽기/쓰기 오류 발생 시 안내 메시지 출력

## 6. 클래스 구조

### 6-1. Quiz 클래스

#### 역할
개별 퀴즈를 표현하는 클래스

#### 속성
- `question`: 문제
- `choices`: 선택지 리스트
- `answer`: 정답 번호

#### 메서드
- `display()`: 문제 출력
- `is_correct()`: 정답 확인
- `to_dict()`: JSON 저장용 변환
- `from_dict()`: JSON → 객체 변환

---

### 6-2. QuizGame 클래스

#### 역할
전체 게임 흐름 관리

#### 속성
- `quizzes`: 퀴즈 목록
- `best_score`: 최고 점수
- `state_file`: 저장 파일명

#### 메서드
- `get_default_quizzes()`
- `load_state()`
- `save_state()`
- `get_text_input()`
- `get_valid_number_input()`
- `show_menu()`
- `play_quiz()`
- `add_quiz()`
- `list_quizzes()`
- `show_best_score()`
- `run()`
